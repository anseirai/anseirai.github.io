Introduction

mybatis的动态查询语句主要有一下几种：

  • if
  • choose
  • where
  • trim
  • set
  • foreach

用来判断某个条件是否满足；当条件满足时，if包含的sql才会生效。

用户update语句，主要功能时忽略以逗号结尾的set语句后面的逗号。另外，set内所有东西，不能全为空。
