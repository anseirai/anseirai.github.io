Introduction

在eclipse打开的情况下，在git上切换分支时，出现下面错误
Unlink of file 'XXXXXXXXX' failed. Should I try again? (y/n)

出现原因

git的工作空间的文件正在被其他程序使用，如果想切换分支，git就会阻止改文件移出或移入git的工作空间。只要关闭所有可能打开使用这些文件的程序即可。

解决方法

把eclipse关了，再切分支就不会发出警告了。
